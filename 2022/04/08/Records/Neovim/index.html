一、iPad端：Termius App

二、服务器：腾讯云服务器（CentOS 7）

1、安装 Tmux：终端复用器

安装前注意参考官方wiki文档

安装方法有如下三种：

  • 直接使用 yum install tmux 安装官方库提供的版本：在 CentOS 中，该版本较旧，无法使用 oh-my-tmux 进行界面美化。
  • 下载源文件，利用官方提供的脚本进行安装：具体步骤参考官方wiki即可。
  • 安装第三方库，从第三方库中获取 RPM 包并安装：在文档中，提及了使用 Galaxy4 第三方库的具体方法。

具体细节可参考Galaxy4库的网站说明

虽然 Galaxy4 库中 Tmux 的版本并非最新版，但已满足需求，为求安装简便，选择使用第三种方法：

在一些 SSH 客户端使用 Tmux 可能无法显示真彩色，如 iPad 上的 Termius。

出现颜色偏差的原因：在进入 Tmux 后使用 echo $TERM，显示screen-256color；而在不使用 Tmux 时输入该命令显示 xterm-256color。

3、 安装Neovim：基于Vim进行升级的代码编辑器

安装前注意参考官方wiki文档

安装方法主要有如下两种：

  • 通过第三方库 epel 使用 yum 包管理命令安装：版本较旧。
  • 通过编译源码进行安装：可安装最新版本，推荐。

此处使用第二种方法进行安装，但在该方法的基础上有更加便捷的衍生方法。

对于该方法，安装前注意参考官方wiki文档中的说明
